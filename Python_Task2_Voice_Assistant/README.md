Description:
A voice assistant programmed using Python that recognizes voice commands and performs various tasks.

Features:
- Voice recognition
- Text-to-speech output
- Play songs on YouTube
- Tell current time
- Tell current date
- Open Google Chrome
- Open Notepad
- Open Spotify
- Search on Google
- Exit using Stop command

Libraries used:
- speech_recognition
- pyttsx3
- pywhatkit
- datetime
- webbrowser
- subprocess

Commands used:
-Hello Nova
-Nova open spotify
-Nova search Python programming courses
-Nova open notepad
-Nova play drop dead by olivia rodrigo
-Nova stop
