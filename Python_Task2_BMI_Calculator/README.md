Description:
BMI Calculator is a Python project that calculates the Body Mass Index(BMI) using the person's weight and height.

Formulae used:
BMI = weight(kgs)/height^2(m)
Height(cm->m) = h/100

Features:
>It takes user input for name, weight and height.
>Converts height from cms to ms.
>Calculates BMI.
>Prints all the user info.
>Also categorise the weight as Underweight, Normal weight, Overweight, Obese.

Technology used:
PYTHON
